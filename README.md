#GiddyOS

GiddyOS is an operating system made with Rust. It uses the information found at the [osdev wiki](http://wiki.osdev.org/Main_Page) and [os.phil-opp.com](http://os.phil-opp.com/).

###First Boot View:

![first-boot]

###Features
* Grub2 as the bootloader
* A simple kernel
